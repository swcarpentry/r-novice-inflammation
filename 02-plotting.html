Plotting

The mathematician Richard Hamming once said, "The purpose of computing is insight, not numbers," and the best way to develop insight is often to visualize data. Visualization deserves an entire lecture (or course) of its own, but we can explore a few of R's plotting features.

Let's take a look at the average inflammation over time. Recall that we already calculated these values above using apply(dat, 2, mean) and saved them in the variable avg_eye_aneurisms. Plotting the values is done with the function plot.

plot(avg_eye_aneurisms)

Above, we gave the function plot a vector of numbers corresponding to the average aneurism count per quadrant across all patients. plot created a scatter plot where the y-axis is the average aneurism counts and the x-axis is the order, or index, of the values in the vector, which in this case correspond to the 4 quadrants of the eye.

Let's have a look at two other statistics: the maximum and minimum inflammation per day.

max_eye_aneurism <- apply(dat[,6:9], 2, max)
plot(max_eye_aneurism)

min_eye_aneurism<- apply(dat[,6:9], 2, min)
plot(min_eye_aneurism)

Defining x and y axes

So far we have plotted a single vector of numbers. R automatically uses the position of the value as the x axis variable. We can explicity define both the x and y axis.

plot(y=dat$BloodPressure,x=dat$Age)

Plotting against a factor

When plotting against a factor R automatically creates a boxplot. This is often a more useful visualisation.

Key Points

  • Use plot to create simple visualizations.
